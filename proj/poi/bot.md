# 机器人

## PoiQQBot

*初代 QQ 机器人*  

Github: <https://github.com/PoiCraft/PoiQQBot-Py>

这是 PoiCraft 的第一个开源项目, 使用 `Python3` 编写, 通过 `酷Q+cqhttp+nonebot` 实现机器人功能. 支持 随机传送, 检查服务器状态 等功能.
